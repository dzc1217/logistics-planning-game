# 🚚 物流规划小游戏 - 本地运行指南

## 🎮 游戏简介

物流规划小游戏是一款教育类游戏，帮助玩家学习不同运输方式的特点和适用场景。玩家需要根据货物特性、客户需求和路线条件选择最合适的运输方式（航空、铁路、水路、公路、管道）。

## 🚀 本地运行方法

### 方法一：直接打开HTML文件（推荐）

1. **下载所有文件**到本地文件夹
2. **双击 `index.html`** 文件
3. 游戏将在浏览器中自动打开

### 方法二：使用HTTP服务器

#### Windows系统：
```cmd
# 方法1：使用Python（如果已安装Python）
python -m http.server 8000

# 方法2：使用Node.js（如果已安装Node.js）
npx http-server -p 8000

# 方法3：使用VS Code的Live Server插件
# 在VS Code中打开index.html，右键选择"Open with Live Server"
```

#### macOS/Linux系统：
```bash
# 使用Python
python3 -m http.server 8000

# 或使用Node.js
npx http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 🎮 游戏操作说明

### 基本操作：
1. **开始游戏**：点击"开始游戏"按钮
2. **选择工单**：点击左侧工单列表中的任务
3. **选择运输方式**：点击中间合适的运输卡片
4. **完成任务**：完成所有工单分配后点击"完成任务"
5. **查看反馈**：查看详细的成绩分析和改进建议
6. **结束游戏**：随时点击右上角"结束游戏"按钮

### 游戏特色：
- ✅ **6个真实场景**：基于实际物流需求设计
- 📊 **即时反馈**：每个选择都有详细分析
- 🎯 **智能评估**：自动判断运输方式选择的正确性
- 💡 **教育价值**：学习不同运输方式的特点和适用场景

## 🐛 常见问题

### Q: 游戏无法正常显示？
A: 确保使用现代浏览器，建议使用Chrome或Firefox的最新版本。

### Q: 按钮点击无反应？
A: 检查浏览器控制台是否有错误信息（按F12打开开发者工具）。

### Q: 本地文件无法运行？
A: 某些浏览器对本地文件有安全限制，建议使用HTTP服务器方法运行。

### Q: 如何重新开始游戏？
A: 在游戏结束界面点击"重新开始"按钮，或刷新页面。
